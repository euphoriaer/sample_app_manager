# QuickUse

## Windows 部署:

服务器解压缩,命令控制行启动
1.需要安装.Net7.0 https://dotnet.microsoft.com/zh-cn/download/dotnet/7.0

1.资源管理器地址前面输入cmd 开启命令行同时切换到当前目录

2.SampleAppManager -urls http://10.0.0.125:5000
